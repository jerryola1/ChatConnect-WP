Easily integrate AI chatbots and WhatsApp chat into your WordPress site with WP ChatConnect.

== Description ==
WP ChatConnect is designed to seamlessly embed AI chatbots and WhatsApp chat links into your WordPress site. Whether you're looking to enhance user interaction, provide better customer support, or boost engagement on your site, this plugin makes it easy. It's versatile, user-friendly, and ideal for businesses, bloggers, and e-commerce platforms.

With WP ChatConnect, you can:
- Embed any AI chatbot script into the header of your WordPress site.
- Add a clickable WhatsApp chat icon for direct communication with visitors.
- Customize placement and appearance of the WhatsApp chat icon.
- Enhance user experience with efficient, real-time interaction.

== Installation ==
1. Upload `wp-chatconnect` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the WP ChatConnect settings page to configure your AI chatbot script and WhatsApp number.

== Frequently Asked Questions ==
= Can I use any AI chatbot script with this plugin? =
Yes, WP ChatConnect is designed to work with any embeddable AI chatbot script.

= How do I add my WhatsApp number? =
Go to the plugin settings, and you can add your WhatsApp number there. It will generate a clickable icon on your site.
